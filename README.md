# Farming for pime

#### farming prototype for pime (new name for game_gui)

###### tui for now.

#### keys
wasd - select place
h - show help message
q - quit
p - plant
i - show inventory
b - break (harvest) crop
g - get watering can
S - set watering can amount to 500
r - refill watering can
W - water crop with watering can
